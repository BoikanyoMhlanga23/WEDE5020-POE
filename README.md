**Project Structure**

- `index.html`: Homepage
- `about.html`: About / team and lightbox images
- `services.html`: Services offered
- `contact.html`: Contact form and map embed
- `enquiry.html`: Enquiry / ticket form
- `message.html`: Simple thank-you page shown after submit
- `stylesheet.css`: Main styles
- `javacript.js`: Client-side scripts (lightbox, form validation, repair-tickets)
- `images/`: Image assets
